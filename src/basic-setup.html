Basic Setup

Basic Setup required for installing LaraPass v1 on your server.

  • Requirements
  • Setup
  • VPS Server
  • VPS Manager

Requirements

  • PHP 7.1.3 (with Extensions)
  • NGINX or Apache
  • MySQL or MariaDB
  • SSH Access (Recommended)

Setup

{primary} LaraPass v1 works best on a VPS with SSH Access.

Below are some of the reputable and amazingly cheap VPS Providers -

Digital Ocean, Vultr, Amazon AWS

Since these VPS provide raw server with resources, we will need a server manager to run our PHP applications such as LaraPass v1. Some of the suggested are -

Ploi, RunCloud, Laravel Forge, Cloudways

All of these service have slightly different pricing and features, so you can check them out and decide for your self. For our demo, we will be using our best setup & most for our money i.e., Digital Ocean with Ploi.

We also recommend downloading & installing Bitvise SSH Client or any other of your choice so you can connect to your server easily.

{warning} While it will work, we do not recommend running LaraPass v1 using Shared Hosting as Shared servers are not Secure and may create issues down the line.

If you are still interested in using the cheaper Shared Hosting services, then we recommend using A2Hosting and use coupon code SOFAST! for 51% Off.

{success} Visit the cPanel Installation Guide Page for details on how to successfully install LaraPass v1 on cPanel.

VPS Server

Setting up a VPS server provided by Digital Ocean or any other from the above list is very simple and straight forward. These steps are general and are not specific to LaraPass v1. If you already know how to provision a DO server, then just skip this section. If you are using Cloudways, then you don't need to register at DO, they will handle everything (albeit at an extra premium).

Step # 1 - Visit DigitalOcean.com
Step # 2 - Create an account. Use coupon/promo code DO10 to get Free Credit to Launch your first server.
Step # 3 - Go to Create Droplets. Select Distritutions/OS as Ubuntu 18.04 x64.
Step # 4 - Select any size as per your budget, we recommend the $10/mo - 2GB Droplet Size & Select a region as per your desire.
Step # 5 - Choose a Hostname and Click Create and wait for the server to get provisioned.
Step # 6 - Once the server is provisioned, you will get an Email with the Server IP & the root password.
Step # 7 - If you haven't already, then you need to point your Domain Nameservers to DigitalOcean.
Step # 8 - Go to Networking -> Domains and Add your Domain.
Step # 9 - Lastly, create 2 new records for your domain as listed below:

 Hostname : @
                        Will direct to : your-server-ip-address
                        TTL : 1800

                        Hostname : www
                        Will direct to : your-server-ip-address
                        TTL : 1800

VPS Manager

A VPS always comes as a barebones pc, as such in order to run web applications or websites on them we need to use a VPS Manager. For LaraPass v1 we will be going with Ploi because unlike others, it comes with a Free Option (along with Free SSL) which allows you to manage your LaraPass v1 site for Free. For more details visit Ploi.io. These steps are general and not specific to LaraPass v1. If you already know how to link your VPS with a Manager, then just skip this section.

Step # 1 - Go to Ploi.io & Sign Up.
Step # 2 - Click on Create Server.
Step # 3 - Click on Custom VPS under Select Provider.
