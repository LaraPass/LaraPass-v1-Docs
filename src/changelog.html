Version 1.4.2 | 03-Jul-2020

Added Login/Register Page Background Header Color Options in Admin Settings.
Added Inner Pages Background Header Color Options in Admin Settings.
Fixed Minor Typos.

Version 1.4.1 | 02-Jul-2020

Added Application URL input option in Admin Settings.
Added reCaptch Setup Incomplete Alert on Registration Page.
Fixed Minor Typos.

Version 1.4.0 | 01-Sep-2019

Added Ignition Error Handler by Facade. Now debug and test the LaraPass v1 App with beautifully represented & detailed error pages.
Added Database variables to the settings file. Can now be changed after installation from the Admin menu.
Added User Deletion. Users can now mark their account for deletion.
Added Email Notifications - Users will now get email notifications for - Marking Account for Deletion, Cancelling Account Deletion, Exporting Accounts, Promoting/Demoting Users.
Added Ability to Promote/Demote Users to/from Admin Status.
Added Ability to change Application Environment from Admin Settings Page (Local Mode Enables Debugging).
Added Pop-up confirmation when deleting Quick/Account Notes.

Updated Icons on the Dashboard and Account Cards.

Fixed Caching issue with the Installer.

Version 1.3.0 | 15-Aug-2019

Added Google Invisible reCAPTCHA - Additional layer of protection from bots.
Added Custom Categories - Now Add, Delete, Activate & Deactivate your own custom categories.
Added Custom Error Pages.
Added Ability to disable 2-Step without the Google Authenticator App.
Added Ability to change the default membership name from Admin Settings.
Added Ability to remove ban/un-ban a user account that has been banned but not permanently deleted.
Added Check for PHP Zip Extension during installation.

Updated Account export option now disabled unless you have at-least 1 account in the vault.
Updated Account export file - added header row for better display of exported data.
Updated Accounts inside protected folders now display a hidden message when exported.
Updated User Profile - added validation check on date of birth to prevent users from selecting a future date.
Updated Privacy & Terms page to reflect user's domain accurately.

Fixed Mobile Navbar Avatar display issue.
Fixed Bug preventing "Add New Account" modal from appearing inside a folder.
Fixed A UI Bug with Account View Modal.
Fixed Site logo in hamburger (mobile) menu.
Fixed A few typos.
Removed PWND Validator. Developing an in-house API for better control over validations. Will re-introduce once ready.

Version 1.2.1 | 11-Feb-2019

Fixed Privacy & Terms Page Links Issue on Login/Register Page.
Fixed A typo in the Auto Updater File (Requires users to manually update LaraPass v1 to v1.2.1 to fix it).

Version 1.2.0 | 05-Feb-2019

Added Account Notes - Encrypted secure notes for each individual account.
Added Breadcrumbs - For easy navigation throughout the application.
Added Password Protection for Folders - Additional password protection for individual folders.
Added Privacy Policy Page - Use the build-in privacy policy page by selecting default in settings.
Added Terms of Service Page - Use the build-in terms of service page by selecting default in settings.
Added Icons for Folders - Folders can be further distinguished using unqiue font-awesome icons.
Added PWNED Validator - Checks and validates passwords against known compromised keywords and blocks the use of any password that has been compromised a minimum number of times.

Updated Admin Overview Page - Added Search, Pagination & Sorting to the User's Table.
Updated Auto-Updater - Updated settings in accordance with v1.2.0.

Fixed Reported issue of user quick menu not displaying as intended on the Dashboard page.
Fixed Reported error when visiting /logout route as a guest user.
Fixed Reported issue of the error handler throwning sensitive app info when debugging (limited to local/debug-on environments).

Version 1.1.5 | 28-Jan-2019

Updated LaraPass v1 Auto-Updater - Prep for v1.2.0 Release and auto-update process.

Version 1.1.0 | 08-Nov-2018

Added Maintenance Mode with Custom Messages.
Added Update Manager - One-Click Tool for easy LaraPass v1 Updates.
Added BackUp Manager - Take full system backups periodically (stored locally).
Added Private Mode - You can now set your app to Private Mode. Only admin can create new users in Private Mode.
Added Admins can now add users directly from the Admin Overview Page. Supplements the Private Mode.
Added Mailer Settings - You can now setup your mailer (for transactional emails) from the Admin Settings page.
Added Cookie Consent - Users will be asked to consent for storing cookies (in compliance of EU Laws).

Updated Admin can now permanantly delete users' (need to Ban them first).
Updated Admin Overview Page now displays the User's "Email Verified At" Column.
Updated Rearranged Admin Overview & Settings Page.

Fixed Default Admin account not displaying avatar.
Fixed IP Logs page issue for Admin User.
Fixed A license file issue.

Version 1.0.2 | 06-Nov-2018

Fixed A bug that was preventing encrypted Support PIN save.
Fixed An issue with Mailer Settings.

Version 1.0.1 | 31-Oct-2018

Fixed A bug that was preventing system settings from being saved correctly

Version 1.0.0 | 30-Oct-2018

Initial Release on Codecanyon
